Definition

\(f:R^n\rightarrow{}R\) is convex if dom\(f\) is a convex set and \[f(\theta{}x+(1-\theta)y)\leq\theta{}f(x)+(1-\theta)f(y)\] for all \(x,y\in\) dom\(f\), \(0\leq\theta\leq1\)
